left recursion (directs and indirects) remotion algorithm

1.	organize nonterminals in some order A1, A2, …, An
2.	for i from 1 to n {
3.		for s from 1 to i - 1 {
4. 			replace each production
				Ai -> As γ with Ai -> δ1γ|δ2γ|...|δkγ, where 
				As -> δ1|δ2|...|δk are all current productions to As
5.		}
6.	// delete any imediate recursion over Ai
7.	replace each production
		Ai -> Ai α1 |...|Ai αn |β1 |...| βm with Ai -> β1 Ai1 |...|βm Ai1, Ai1 -> α1 Ai1 |...| αn Ai1 | Ɛ, where
		α1,...,αn,β1,...,βm do not start with Ai
8.	}
